  • descriptionO que é o Tinp?

    É um site que permite ao usuário desenhar livremente utilizando a ferramenta canvas, do HTML 5.0.
    Também permite ao usuário votar em imagens feitas por outros usuários, as mais votadas são mostradas num mural.
    Foi construindo utilizando o framework Materialize para o Front-End, PHP e MariaDB para o Back-end.
